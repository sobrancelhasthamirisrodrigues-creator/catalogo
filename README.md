# Thamiris Rodrigues - Catálogo de Micropigmentação

Catálogo online profissional showcasing os trabalhos de micropigmentação de Thamiris Rodrigues.

## 🎨 Características

- **Design Moderno:** Interface elegante com fundo blur e animações suaves
- **Performance Otimizada:** 60fps constante, hardware acceleration, lazy loading
- **Fully Responsive:** Perfeito para desktop, tablet e mobile
- **Catálogo Visual:** 9 trabalhos em sequência ordenada
- **Download PDF:** Gera catálogo completo em PDF com todas as imagens
- **Animações 2026:** Efeitos modernos de texto e interações

## 🚀 Tecnologias

- **HTML5** Semântico e otimizado
- **CSS3** com variáveis, glassmorphism e performance
- **JavaScript ES6+** com animações avançadas
- **jsPDF** para geração de catálogos em PDF
- **Lazy Loading** para carregamento otimizado

## 📁 Estrutura

```
├── index.html          # Página principal
├── style.css           # Estilos e animações
├── script.js           # Funcionalidades e efeitos
```

## 🌐 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/sobrancelhasthamirisrodrigues-creator/catalogo.git
   ```

2. **Navegue até a pasta:**
   ```bash
   cd catalogo
   ```

3. **Abra o site:**
   - Dê duplo clique em `index.html`
   - Ou use um servidor local para melhor performance
